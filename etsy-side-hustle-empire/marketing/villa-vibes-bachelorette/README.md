# Villa Vibes Bachelorette Templates

This folder contains ready-to-use Etsy listing and Canva copy templates for a "Villa Vibes" bachelorette printable bundle.

## Files
- `ETSY_LISTING_TEMPLATE.md` — SEO title options, full listing description, tags, materials, and attributes.
- `CANVA_TEMPLATE_COPY_DECK.md` — paste-ready text for signs, invitations, itinerary, games, paddles, and bar cards.

## Recommended workflow
1. Duplicate and edit the copy with each bride's name, city, and date.
2. Build templates in Canva using US Letter + A4 versions.
3. Export print files as PDF (300 DPI).
4. Add Canva edit-link instructions in your delivery PDF.
