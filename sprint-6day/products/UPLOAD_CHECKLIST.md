# Etsy Upload Checklist — 5 Products
**Time estimate: 8-10 minutes per listing**

---

## HOW TO UPLOAD (quick reference)

1. Go to **etsy.com → Shop Manager → Listings → Add a listing**
2. Upload the `.xlsx` file (digital download)
3. Copy the listing **Title** from `LISTING.md`
4. Set the **Price** from `LISTING.md`
5. Copy-paste the full **Description** from `LISTING.md`
6. Add all **13 Tags** from `LISTING.md`
7. Set **Category**: Templates & Digital Files → Spreadsheets (or Planners)
8. Set **Type**: Digital download
9. Set **Instant download**: Yes
10. Upload **5 listing photos** (Canva mockups — see `CANVA_MOCKUP_GUIDE.md`)
11. **Publish**

## SETTINGS FOR ALL LISTINGS

- **Listing type:** Digital download
- **Instant download:** Yes (buyers get it immediately)
- **Format:** Spreadsheet (.xlsx — works in Google Sheets + Excel)
- **Quantity:** Unlimited (it's digital — set to 999)
- **Returns:** No returns/exchanges (standard for digital)
- **Language:** English
- **Ships from:** N/A (digital)
