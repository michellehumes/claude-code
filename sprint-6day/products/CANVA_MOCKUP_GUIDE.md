# Canva Listing Image Guide — All 5 Products
**Create 5 images per product. Size: 2000x2000px (square) or 2700x2025px (4:3)**

## CANVA SETUP (do once)

1. Go to canva.com → Create a design → Custom size → 2000 x 2000 px
2. Search for "laptop mockup" in Elements — use a clean silver MacBook frame
3. Save your base template — duplicate it for each product

## QUICK MOCKUP METHOD (15 min per product)

1. **Get screenshots**: Open each .xlsx in Google Sheets, take screenshots of the key tabs
2. **Open Canva**: New design → 2000x2000px
3. **Add MacBook frame**: Elements → search "laptop mockup" → use the clean silver one
4. **Drop screenshot into screen**: drag your screenshot inside the laptop screen area
5. **Add text overlay**: product name + tagline in matching font
6. **Export as PNG**: Download → PNG → high quality
7. Repeat for 5 photos per product

**Time saver**: Use a Canva template. Search "Digital Product Mockup" — hundreds of free ones.
