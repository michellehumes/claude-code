- **Page Size:** 8.5 x 11 inches (US Letter)
- **Margins:** 0.75-1 inch all sides
- **Line Height:** 1.5 for body text
- **Input Line Height:** 28-36px
- **Checkbox Size:** 14 x 14px with 1px border

## Canva Recreation Guide

### Step 1: Create New Document
1. Open Canva and create a custom size: **8.5 x 11 inches**
2. Set background color to `#F8F6F2`

### Step 2: Add Fonts
Search for and add these fonts:
- **Playfair Display** (headers)
- **Montserrat** (body text)

### Step 3: Create Design Elements

#### Divider Lines
- Draw a rectangle: Width 100%, Height 1px
- Fill color: `#CFC6BC`
- No stroke

#### Accent Lines
- Rectangle: Width 40-60px, Height 2px
- Fill color: `#B7C1B2`

#### Input Lines (for writing)
- Rectangle: Variable width, Height 1px
- Fill color: `#CFC6BC`
- Position at bottom of input area

#### Checkboxes
- Square shape: 14 x 14px
- No fill, 1px stroke
- Stroke color: `#CFC6BC`
- Corner radius: 2px

#### Highlight Boxes
- Rectangle with fill: `#B7C1B2` at 15% opacity
- Optional: 2px left border in `#B7C1B2`

### Step 4: Text Styling

**For Section Headers (ALL CAPS):**
```
Font: Montserrat
Weight: Medium (500)
Size: 14pt
Letter spacing: 200
Transform: UPPERCASE
Color: #2E2E2E
```

**For Page Titles:**
```
Font: Playfair Display
Weight: Regular
Size: 28pt
Letter spacing: -20
Color: #2E2E2E
```

**For Body Text:**
```
Font: Montserrat
Weight: Regular
Size: 11pt
Letter spacing: 0
Line height: 1.5
Color: #2E2E2E
```

**For Labels:**
```
Font: Montserrat
Weight: Medium (500)
Size: 9pt
Letter spacing: 100
Transform: UPPERCASE
Color: #5A5A5A
```

### Step 5: Page Footer
- Left side: Page section name (uppercase, Montserrat 10pt)
- Right side: Page number
- Position: 0.5 inches from bottom

## Export Settings

### For Print PDF
1. File > Download
2. Select: PDF Print
3. Flatten PDF: Off
4. Crop marks and bleed: Off (unless printer requires)

### For iPad/GoodNotes
1. File > Download
2. Select: PDF Standard
3. This creates a lighter file optimized for digital use

## Design Philosophy

These templates follow key principles for Etsy success:

1. **White Space is Premium** - Generous margins and spacing signal quality
2. **Functionality Over Decoration** - Every element serves a purpose
3. **Print + Digital Versatility** - Works beautifully in both formats
4. **Timeless Over Trendy** - Neutral palette appeals to broad audience
5. **Cohesive System** - Both products share design DNA for bundle appeal
