How to Print Your Bundle

  1. Open any template file in your web browser (Chrome recommended)
  2. Press Ctrl+P (or Cmd+P on Mac) to open the print dialog
  3. Set Destination to "Save as PDF" for a digital file, or select your printer
  4. Set Margins to "None" for edge-to-edge printing
  5. Enable "Background graphics" in More Settings to keep all colors
  6. Use Letter size (8.5" x 11") paper
  7. For best results, print on cardstock (especially signs, dare cards, tent cards, paddles)
  8. For customizable fields (dashed lines), write in by hand after printing, or edit the HTML before printing
