1. Log in to your Etsy seller account at **etsy.com/your/shops/me/tools/listings**

### STEP 4: Upload Digital Files

In the "Digital files" section:
1. Click "Upload a file"
2. Upload all 6 spreadsheet files:
   - `Weekly_Meal_Planner_2026.xlsx`
   - `Monthly_Meal_Calendar_2026.xlsx`
   - `Recipe_Collection_Tracker.xlsx`
   - `Grocery_Budget_Tracker_2026.xlsx`
   - `Pantry_Inventory_Tracker.xlsx`
   - `Meal_Prep_Planner.xlsx`

**Important:** Etsy allows up to 5 digital files per listing. If you hit the limit, you can:
- Option A: Create a ZIP file containing all 6 spreadsheets and upload the single ZIP
- Option B: Split into two listings and offer as a combined purchase

**Recommended approach:** Create a ZIP file named `Meal_Planner_Bundle_2026.zip` containing all 6 .xlsx files.

### STEP 6: Description

Copy the full description from `etsy_listing_copy.md`. Paste it into the description field. Etsy supports basic formatting -- use line breaks and capital letters for headers since Etsy does not support markdown.

**Formatting tips for Etsy:**
- Use ALL CAPS for section headers
- Use dashes (--) for bullet points or Etsy's built-in bullet feature
- Add line breaks between sections for readability
- Emojis can help with visual scanning but use sparingly
