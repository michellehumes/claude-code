## Launch Checklist (Per SKU)

### Pre-Launch
- [ ] All assets created and quality-checked
- [ ] Files organized in clear folder structure
- [ ] Instruction guide complete
- [ ] Canva template links tested
- [ ] PDF print quality verified at 300 DPI
- [ ] Listing title optimized (140 chars, keywords front-loaded)
- [ ] 13 tags selected (keyword research verified)
- [ ] Description written (all sections complete)
- [ ] FAQ section added (5+ questions)
- [ ] Pricing set (tier structure confirmed)
- [ ] All mockup images created (minimum 5, ideal 10)
- [ ] First image is a lifestyle/hero shot (not a grid)
- [ ] Cross-sell links prepared (related listings)
- [ ] Launch discount set (25% for 2 weeks)

### Launch Day
- [ ] Listing published on Etsy
- [ ] Listing published on Gumroad (if applicable)
- [ ] Download tested from buyer perspective
- [ ] All links working (Canva templates, instructions)
- [ ] Social media announcement (if channels exist)
- [ ] Etsy Ads activated ($1â€“$3/day test budget)

### Post-Launch (Week 1)
- [ ] Check views and conversion daily
- [ ] Monitor for first sale
- [ ] Check for customer questions / issues
- [ ] Verify no file delivery problems
- [ ] Respond to any reviews within 24 hours

### Post-Launch (Week 2â€“4)
- [ ] Analyze keyword performance
- [ ] Adjust tags if needed
- [ ] Remove launch discount (return to full price)
- [ ] Start building next SKU from this vertical
- [ ] Document learnings for future launches
