# Portfolio Expansion Roadmap

## Overview

This roadmap outlines the SKU scaling strategy from initial 7 launches to 20+ SKUs within 90 days. Built on the core principle: **60–70% structure reuse across adjacent themes with minimal incremental production cost.**

| 14 | Zodiac Friendship Compatibility Kit | Astrology | 80% | Reframe romance → friendship |

### Production Efficiency

| Reuse Level | Meaning | Avg Production Time |
|-------------|---------|-------------------|
| 80% reuse | Language swap only (cat vs. dog) | 2–3 hours per SKU |
| 70% reuse | Palette + minor illustration swap | 4–6 hours per SKU |
| 65% reuse | New theme + illustrations, same structure | 8–12 hours per SKU |
| 50% reuse | New art content, same format/sizes | 16–24 hours per SKU |
