## Design Specifications (Applied to All Signs)

### Layout Principles
- Center-aligned text
- Generous margins (minimum 15% of sign width on each side)
- Title in script or serif, body in clean sans-serif
- Decorative elements: corner watercolor washes, not borders
- One small coastal illustration element per sign (shell, coral, eucalyptus)

### Font Application
- **Main line:** Cormorant Garamond Bold, proportional to sign size
- **Body copy:** Montserrat Light, 40–50% of header size
- **Couple names:** Playlist Script (or elegant script), slightly larger than header

### Color Application Per Sign
- Primary text: Charcoal (#3A3A3A) or Dusty Blue (#A4C3D2) for headers
- Body text: Charcoal (#3A3A3A)
- Decorative: Muted Gold (#C9A96E) for rules/dividers, Sand (#E8DDD3) for washes

### Print Specs
- All signs: 300 DPI, PDF, CMYK
- Include crop marks for signs larger than 11x14
- Standard sizes: 5x7, 8x10, 11x14, 16x20, 18x24, 24x36
- Home-print optimized for 5x7 through 11x14
- Professional print recommended for 16x20+

## Mockup Direction (Signs)
- Acrylic easel display on reception table
- Wooden easel at ceremony entrance
- Framed in simple gold or white frame
- Beach/garden setting in background, soft focus
