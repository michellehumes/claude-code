- Gender-neutral throughout — no pink/blue defaults

- **Canva:** Editable template links for invitation + key editable pieces
- Cut guides included for toppers, tags, and props

## Editable Name Personalization Logic

All pieces containing the child's name use:

1. **Canva editable fields** — buyer changes name directly
2. **Clear text instruction sheet** — step-by-step with screenshots
3. **Font matching guide** — exact fonts to install/use
4. **Test print page** — quick reference to verify before full print

For pieces without Canva access (PDF-only buyers):
- Name field left as "[Child's Name]" placeholder
- Instructions to use free tools (Adobe Reader form fields or Canva free tier)

**Upsell opportunity:** "Want us to personalize all pieces? Add our Name Customization Package — $15"
