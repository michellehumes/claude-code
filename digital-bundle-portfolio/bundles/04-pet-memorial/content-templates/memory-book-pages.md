# Content Template: Memory Book Pages — Dog Remembrance Tribute Kit

## Tone Guide

Every word in this kit must feel like a quiet, compassionate hand on the shoulder. Never clinical. Never transactional. This is sacred content for people in grief. Write with warmth, dignity, and restraint.

## Universal Design Specs (All Memory Pages)

### Color Palette
- Background: Soft Linen (#F7F4EF)
- Text: Warm Charcoal (#4A4540)
- Accents: Dusty Gold (#C9B97A), Muted Sage (#B8C5B2)
- Lines: Light Warm Gray (#D5D0C8)

### Typography
- Page titles: Cormorant Infant Semibold, 24pt
- Prompts: Cormorant Infant Italic, 12pt
- Fill-in text areas: Quicksand Light, 11pt
- Quotes: Cormorant Infant Italic, 14pt
- Name (script): Allura, 36pt+

### Print Specs
- Size: 8.5x11 (letter) — all pages
- Resolution: 300 DPI
- Color: sRGB
- Format: PDF
- No bleed (home printing)
- Margins: 0.75" all sides minimum
- Paper recommendation: 80–100lb text weight, warm white or cream

### Assembly Note
Include a cover sheet with:
- Suggested page order
- Printing tips (paper weight, single-sided recommended)
- Binding suggestions (3-ring binder, spiral, or booklet)
- "Take your time with these pages. There's no rush."
